## 這可能是一隻 Discord 機 雞 人

這是一個雞於 Node.js 的 Discord 機雞人，採用我們覺得很厲害的屎山代碼。  
説到屎山，屎山不是一天堆成的，就像羅馬不是一天造成的，可是羅馬也不是一天能逛完的，所以當你看到這座屎山，你要心懷敬畏，因為每一坨都有它存在的理由，哪怕那個理由只是「當時覺得這樣寫很帥」。  
注意，這是一個幹話密度極高的文檔，如果你不喜歡廢話，那你八成是來對地方了；來都來了，不如就看看，看完你會發現，看跟沒看其實差別不大，但心情可能會微妙地好一點點，這就是廢話的魔法。

### 這應該是開始？

#### 1. 設定依嗯微
你應當優雅地複製 `.env.example` 並重新命名為 `.env`，然後填入必要的資訊，必要的意思就是如果不填就會出事，出了事就會後悔，後悔就來不及了。  
為什麼叫依嗯微呢？因為 `.env` 唸快一點就是依嗯微，如果你堅持要唸「點 E N V」，這也是一種浪漫，只是比較長。
```bash
# 用高雅人士系統，也是 cp .env.example .env
cp .env.example .env
```
所需的變數包含：
- `TOKEN`: 你的機雞人 Token。Token 就像雞腳上的腳環，沒有它，沒人知道這隻雞是誰家的，伺服器會直接假裝不認識你。

#### 2. 下載 node 該死的包包
包包就是依賴，依賴就是你需要別人寫好的程式碼，這樣你就不用自己造輪子。但輪子造多了，車庫就滿了，車庫滿了就要換更大的車庫，人生就是這樣。
```bash
# 在 root 目錄下執行
npm install
```
這行指令會把`拉尼亚凯亚超星系团，室女座星系团，本星系群，银河系，猎户座旋臂，太阳系，太阳，第三行星，地球`的程式碼都搬到你的 `node_modules` 裡。黑洞般的資料夾，裡面什麼都有，也什麼都沒有，因為你永遠不會點進去看。

#### 3. 部署 Discord 最飄撇的斜線指令
在讓你的機雞人起床之前，你還需要把指令註冊到我親愛的 Discord。註冊的意思就是告訴 Discord：「嘿，這隻雞會這些把戲喔！」如果不註冊，機雞人就只會站在原地咯咯叫，而你會納悶為什麼雞不理你。
```bash
# 在 root 目錄下執行
npm run deploy
# 怎麼讓那些斜線指令人間蒸發呢？
npm run clear
```
`deploy` 是部署，`clear` 是清除，就像給雞吃飼料跟幫忙掃雞舍，都是不可或缺的關愛。清除之後指令會消失，但雞還在，只是暫時變成文盲，有需要的話再部署一次即可。

#### 4. 啟動機雞人

##### 你可以使用 dev 模式啟動
`dev` 就是 `develop` 的簡寫，簡寫之所以迷人，是因為可以省下零點幾秒的打字時間，日積月累下來大概可以多喝一杯水。
```bash
# 在 root 目錄下執行
npm run dev
```
##### 你也可以使用 start 啟動（那當然我非常推薦你使用 `start` 因爲比較高大賞）
`start` 就是開始，開始了就希望它不要隨便結束，結束了就是 `stop`，而 `stop` 是 `start` 的相反，如果你沒按 `stop`，它就會一直跑，跑到天荒地老或你的電腦當機為止。
```bash
# 在 root 目錄下執行
npm run start
```
啟動成功後，你的機雞人就會破殼而出，正式成為數位世界的一員。記得對它說聲早安，雖然它不會回你早安，因為它只會執行指令，情感模組還沒裝。
