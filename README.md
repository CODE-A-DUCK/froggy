## 這是可能是一隻 Discord 雞 雞 人

這是一個🐔（哎喲你幹嘛）於 Node.js 的 Discord 雞雞人，採用我觉得很厲害的屎山代碼（多看看我是怎么写屎山代码的吧！你媽了個逼！！！！！！）。

### 這應該是開始？

#### 1. 設定依嗯微
你個傻逼應該複製 `.env.example` 並重新命名為 `.env`，然後填入所謂的必要的資訊：
```bash
# 我用高雅人士系統，應該是 mv .env.example .env
cp .env.example .env
```
所需的變數包含：
- `TOKEN`: 你的雞雞人 Token。

#### 2. 下載你媽的包包👜
```bash
# 在 root 目錄下執行
npm install
```

#### 3. 部署 Discord 最垃圾的斜線指令
在啟動你的雞雞人之前，你他媽還需要將指令註冊到我親愛的 Discord：
```bash
# 在 root 目錄下執行
npm run deploy
# 怎麼清除該死的斜線指令呢？
npm run clear
```

#### 4. 啟動雞雞人

##### 你可以使用 dev 模式啟動

```bash
# 在 root 目錄下執行
npm run dev
```
##### 你也可以使用 start 啟動

```bash
# 在 root 目錄下執行
npm run start
```
